# 🎓 Smart Student Assistant Chatbot 🤖

## 📌 Description
This project is an AI-based chatbot developed using Natural Language Processing (NLP) and Machine Learning (Naive Bayes).  
It can understand user queries and provide useful responses related to study, internships, and placements.

---

## 🚀 Features
- 📚 Study guidance and exam tips  
- 💼 Internship assistance  
- 🎯 Placement preparation  
- 🤖 Smart intent classification using ML  
- ⚡ Confidence-based fallback response  
- 💬 Interactive chat UI with typing effect  

---

## 🧠 Technologies Used
- Python  
- Flask  
- NLTK (Natural Language Processing)  
- Machine Learning (Naive Bayes)  
- HTML, CSS, JavaScript  

---

## ⚙️ How It Works
User Input → NLP Processing → Machine Learning Model → Intent Detection → Response Generation → Output Display  

---

## ▶️ How to Run the Project

1. Install dependencies:
   pip install nltk numpy scikit-learn flask flask-cors
2. Train the model:

python train.py


3. Run the chatbot:

python app.py


4. Open browser:

http://127.0.0.1:5000
